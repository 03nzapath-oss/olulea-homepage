# 画像フォルダ

このフォルダには、ウェブサイトで使用される画像を配置します。

## フォルダ構成

```
images/
├── hero/          # ヒーロー画像（メインビジュアル）
│   ├── main.jpg   # トップページのメイン画像 (推奨: 1920x1080px)
│   └── store.jpg  # 店舗外観の画像 (推奨: 800x800px)
└── gallery/       # ギャラリー画像
    ├── 1.jpg      # サロン内観1 (推奨: 800x800px)
    ├── 2.jpg      # サロン内観2 (推奨: 800x800px)
    ├── 3.jpg      # カットスタイル1 (推奨: 800x800px)
    ├── 4.jpg      # カットスタイル2 (推奨: 800x800px)
    ├── 5.jpg      # シャンプー台 (推奨: 800x800px)
    └── 6.jpg      # 商品・設備 (推奨: 800x800px)
```

## 画像の追加・変更方法

1. **画像を準備**
   - 形式: JPG, PNG, WebP
   - サイズ: 上記の推奨サイズを参照
   - 画質: Web用に最適化されたもの

2. **ファイル名を確認**
   - 既存の画像を差し替える場合: 上記のファイル名と同じ名前を使用
   - 新しい画像を追加する場合: `config/images.ts` で設定を追加

3. **画像を配置**
   - 該当するフォルダに画像ファイルをコピー

4. **動作確認**
   - 開発サーバーを起動して表示を確認
   - ブラウザでキャッシュをクリアして再読み込み（Cmd+Shift+R）

## 注意事項

- **ファイルサイズ**: Web表示用に圧縮してください（目安: 200KB以下）
- **著作権**: 使用する画像の著作権・肖像権に注意してください
- **アスペクト比**: 
  - ヒーローmain: 16:9 または 横長
  - その他: 1:1（正方形）を推奨

## 画像が見つからない場合

画像ファイルが配置されていない場合、自動的にプレースホルダー画像が表示されます。
実際の画像を配置するまで開発を継続できます。
